## Tasks

### Task 1: Add baseline snapshot + delta extraction to PingPongRouter

**Why:** Currently `onTurnComplete` delivers the full viewport snapshot. For relay injection, we need only the NEW output from the current turn (not old prompts, previous history).

**Changes:**
- PingPongRouter.swift: Add `baselineText` property
- In `startCapturingCurrentPane()`, capture baseline from current viewport BEFORE starting timer
- In `processTurnComplete()`, compute delta: strip baseline prefix from currentTurnText
- Deliver only the delta through `onTurnComplete`

### Task 2: Wire relay injection in onTurnComplete callback

**Why:** This is the CRITICAL missing piece. When pane A's turn completes, inject the cleaned output into pane B's stdin via `sendText()`.

**Changes:**
- WorkspaceWindowController.swift `togglePingPong()`:
  - Modify `onTurnComplete` callback to call `sendText()` on the target pane
  - Target = the pane that is about to start capturing (the "listener")
  - Add `\n` to trigger command execution in the target agent

### Task 3: Improve mission control active pane indicator

**Why:** PING-04 requires clear active pane indicator. Current "A speaking..." is functional but could be more prominent.

**Changes:**
- WorkspaceWindowController.swift: Update `onStateChanged` to show directional relay indicator
  - "A speaking..." → "A → B" / "B → A"
- Keep round counter and paused state as-is (already working)
