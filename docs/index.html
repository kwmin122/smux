How It Works

AI 에이전트 합의 시스템을 통한 가장 안전한 자동화 워크플로우

The safest automated workflow through AI agent consensus

01

Install

다운로드하고 앱을 열면 바로 시작. 복잡한 설정 없이 로컬 환경에 즉시 통합됩니다.

Download and open the app to get started instantly. Integrates into your local environment with zero configuration.

02

Configure

Planner와 Verifier 선택, 합의 전략 설정. 프로젝트 성격에 맞는 최적의 AI 조합을 구성하세요.

Choose your Planner and Verifier, set consensus strategy. Configure the optimal AI combination for your project.

03

Run

에이전트들이 자동으로 코드를 작성하고 검증. 다수결 또는 만장일치 원칙에 따라 실시간으로 피드백을 주고받습니다.

Agents automatically write and verify code. They exchange real-time feedback following majority vote or unanimous consensus.
