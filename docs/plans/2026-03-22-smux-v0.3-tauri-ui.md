# smux v0.3 Tauri UI Implementation Plan

> **Goal:** Wrap smux-core in a native macOS app with Tauri v2. Split-pane terminal UI, Focus/Control modes, embedded browser, panel drag-and-drop. cmux-level polish.

**Architecture:** Tauri v2 app (`crates/smux-app/`) with React + TypeScript frontend. Connects to smux-daemon via Unix socket IPC. Terminal panels use xterm.js. Browser panel uses Tauri's WebView.

**Tech:** Tauri v2, React 19, TypeScript, Vite, xterm.js, Tailwind CSS

---

### Task 17: Tauri v2 App Scaffolding

**What:**
- `cargo tauri init` inside `crates/smux-app/`
- Add to workspace
- React + Vite + TypeScript frontend in `crates/smux-app/src/`
- Tauri Rust backend connects to smux-core
- Basic window opens with "smux" title bar
- `cargo tauri dev` works

**Tests:**
- App window opens
- Tauri IPC bridge responds to ping

---

### Task 18: xterm.js Terminal Panels + Daemon Connection

**What:**
- xterm.js component that renders agent output
- Connect to daemon via Unix socket (through Tauri command bridge)
- Tauri Rust command: `start_session`, `attach_session`, `get_events`
- Terminal panel receives and renders AgentOutput events in real-time
- Monospace font, dark theme, auto-scroll

**Tests:**
- Terminal renders text
- Events from daemon appear in terminal

---

### Task 19: Focus Mode (Default)

**What:**
- Left panel: Planner terminal (xterm.js)
- Right panel: Verifier terminal (xterm.js)
- Top status bar: smux FOCUS | pingpong R3/10 Phase 1 | [Tab]
- Bottom shortcut bar: [Tab] Control [i] Intervene [r] Rewind [d] Diff [q] Quit
- Resizable divider between panels
- Blue status bar background

**Tests:**
- Two panels render side by side
- Status bar shows round info
- Divider is draggable

---

### Task 20: Control Mode ([Tab] Toggle)

**What:**
- Three-panel layout: Planner | Mission Control | Verifier
- Mission Control center panel:
  - Round History (R1 ✗, R2 ✗, R3 ✓)
  - Rewind selector
  - Health indicators (Planner ● 62%, Verifier ● 84%)
  - Safety status
  - Git info (branch, files changed, [d] Diff)
  - Event Log
- Orange status bar in Control mode
- [Tab] toggles between Focus and Control

**Tests:**
- Tab key switches modes
- Control panel shows round history
- Health indicators update

---

### Task 21: Embedded Browser Panel

**What:**
- Tauri WebView for browsing localhost apps
- [Cmd+B] toggles browser panel
- URL bar with navigation
- Shows agent-browser E2E test results in real-time
- Integrates with 3-way verification display

**Tests:**
- Browser panel opens with URL
- Navigation works
- Panel can be toggled

---

### Task 22: Panel Drag-and-Drop + Layout Presets

**What:**
- All panels are independent, draggable blocks
- Cmd+1/2/3 for preset layouts (Center/Right/Bottom)
- Cmd+Drag for free positioning
- Cmd+S to save custom layout
- Cmd+F for fullscreen panel
- Panel border drag to resize
- Layout persisted in ~/.smux/layouts/

**Tests:**
- Preset layouts switch correctly
- Custom layout saves and loads
- Panel resize works
