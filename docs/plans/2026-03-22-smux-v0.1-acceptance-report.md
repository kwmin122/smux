## Architecture Decisions Made During v0.1

| Decision | Rationale |
|----------|-----------|
| HeadlessCliAdapter for both Claude and Codex | Both support structured headless output. PTY deferred to v0.2 |
| `--output-format text` instead of `stream-json` | Wire protocol UNVERIFIED. Text sufficient for stop detection |
| Commit-per-round on worktree branch | More reliable than git tag/stash for rewind |
| `git clean -fd` (not `-fdx`) | Preserve .gitignore'd artifacts (build cache) |
| No daemon in v0.1 | Direct execution simpler. Daemon for v0.2 |
| `async_trait` for object safety | Rust native async-in-trait doesn't support dyn dispatch |
